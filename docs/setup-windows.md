# Windows Environment Setup (Coming Soon)

Full Windows setup instructions are being worked on and will be added in a future release.

---

## In the Meantime: Use Claude Projects (Web)

The easiest way to use the SAP ECS R&R Advisor on Windows today is through Claude Projects in your browser — no installation required.

Follow this guide instead: [docs/setup-web-projects.md](setup-web-projects.md)

Claude Projects work on any operating system with a modern web browser.

---

## What Windows Support Will Cover

When the Windows guide is published, it will cover:

- Installing Python 3.10+ via python.org or winget
- Installing pdftotext (Poppler for Windows)
- Creating a virtual environment
- Running setup.py and refresh.py
- Scheduling automatic refresh via Windows Task Scheduler (equivalent to launchd on Mac)
